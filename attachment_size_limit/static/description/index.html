Attachment Size Limit (Document Management System DMS)

This app allow you to specify attachment size limit on users form which will allow you to restric user to upload file/attachment which are greater than size specified for the user(s).

Main Features:

  • Attachment Size Limit set on user form..
  • Only Apply to users who have Attachment Size Limit Group Ticked.
  • Applicable limit only on when user try to add attachment in open chatter and using manage attachment option on developer mode. This limit does not apply to any attachment which comes from websites.
  • For more details please check below screenshots and watch the video.
